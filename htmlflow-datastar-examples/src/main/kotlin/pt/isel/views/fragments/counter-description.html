Counter — Description

  1. On page load, a GET to /counter/events opens an SSE stream that patches the counter display via patchElements whenever the counter value changes.
  2. Clicking Increment sends a POST to /counter/increment — the server increments the counter and the SSE stream emits the updated value.
  3. Clicking Decrement sends a POST to /counter/decrement — the server decrements the counter and the SSE stream emits the updated value.
  4. When the counter reaches 3, the server sends an executeScript SSE event that triggers a browser alert.
